# videos

Tools for working with video footage — AI-assisted clipping, triage, and editing helpers.

## Tools

- **[condor-clipper](condor-clipper/)** — batch-triage a folder of videos into
  condor-only clips. Claude vision confirms the species and scores how striking
  each shot is; `ffmpeg` cuts the keepers and writes a ranked manifest.

Each tool has its own README with setup and usage.
